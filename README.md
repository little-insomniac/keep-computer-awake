### Option 2: From source

Requires [Node.js](https://nodejs.org) and [Git](https://git-scm.com).

```bash
git clone https://github.com/little-insomniac/keep-computer-awake.git
cd keep-computer-awake
npm install
npm start
```

## ✨ Features

- Prevent your computer from sleeping or locking the screen
- Works on Mac, Windows, and Linux
- Lightweight and runs silently in the background
